# Документация по API
## 1. Функционал для администратора системы
### 1.1 Авторизация в системе

  POST /auth/token/login/
    
    Тело запроса содержит имя пользователя и пароль:
    username=admin
    password=admin
    
    Тело ответа:
    {
      "auth_token": "71fe64018947bf8ad9a2e7739fc18613d4e42886"
    }
